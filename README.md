# 📂 프로젝트 소개 
​
안녕하세요! <br/>
**LSM**은 <br/>
교육자는 LMS를 사용하여 강의와 다양한 학습 자료를 업로드하고 <br/>
관리할 수 있습니다!<br/>
교육 대상자들의 정보를 추적하고 사용자 그룹을 관리할 수 있습니다. <br/>
이를 통해 특정 교육 대상자에게 특별한 권한을 부여하거나 
그룹별로 교육을 관리할 수 있습니다. <br/>


​<br/><br/>
온라인을 통해 수강생들에게 교육 전반에 걸친 <br/>
강의 관리를 제공함으로써 <br/>

**효율적이고 편리한 교육 환경을 제공**합니다
